Pagination

Information about pagination is provided in the Link header of an API call. For example:

curl -I "https://gekwetter.com/api/v1/vfonic?page=2"

The -I parameter indicates that we only care about the headers, not the actual content. 

Link: <http://gekwetter.dev/api/v1/vfonic?page=1>; rel="first", <http://gekwetter.dev/api/v1/vfonic?page=1>; rel="prev"
Link: <http://gekwetter.dev/api/v1/vfonic?page=34>; rel="last", <http://gekwetter.dev/api/v1/vfonic?page=3>; rel="next"
